# Desi Dhaaga Website

## Project Overview

Desi Dhaaga is a premium fashion brand website dedicated to timeless, elegant garments for men and women. The site showcases sophisticated collections of high-quality clothing crafted with precision, blending modern design with superior craftsmanship. It aims to inspire customers with stunning visuals, detailed product presentations, and a seamless luxury shopping experience.

The website features:
- Elegant and minimalist design that highlights garment aesthetics, fabrics, and tailoring.
- High-resolution imagery of clothing in real-world styling and lifestyle settings.
- Easy navigation to explore collections for men and women.
- Sections for featured collections, shop, about the brand, and contact.

Website: https://www.srinikajanatextiles.com/

## Features
- Fully responsive design optimized for desktop, tablet, and mobile devices.
- Smooth navigation menu: Home, Men, Women, Shop All, About Us, Contact.
- Hero sections with impactful visuals and taglines emphasizing luxury, craftsmanship, and timeless style.
- Product showcases with high-quality images, detailed descriptions, and styling inspiration.
- Call-to-action elements for inquiries, personalized styling, and customer support.

## Technologies Used
- HTML5
- CSS3 (with modern layouts, subtle animations, and custom elegant styling)
- Vanilla JavaScript (for interactive elements and smooth user experience)
- No external frameworks – pure custom code for lightweight performance and full control

## Installation / Setup (for local development)
Since this is a static site:
1. Download or clone the project folder.
2. Place all files (`index.html`, `about.html`, `contact.html`, `privacy-policy.html`, `terms-conditions.html`, and `img/` folder) in the same directory.
3. Open `index.html` in your browser to view the site locally.
4. For deployment, upload the entire folder to any static hosting platform (e.g., Netlify, Vercel, GitHub Pages, or traditional web hosting).
